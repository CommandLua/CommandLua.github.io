ScenEdit_EventX ()

This function indicates the current event that has been triggered.

Note that EventX() can also be used as a shortcut for ScenEdit_EventX()

Parameters

    None

Returns

Event The triggering event as a wrapper, else a nil is returned.

Example
local event = ScenEdit_EventX()
