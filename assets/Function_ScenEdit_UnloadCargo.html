ScenEdit_UnloadCargo (fromUnit)

This function Unloads all cargo from a unit. This is equivalent to the 'Unload cargo' unit order.

The previous 'cargoList' parameter to this function was not used and has been deprecated. If a cargoList is passed it will be ignored.

Parameters

  • fromUnit string The unit name/guid with cargo

Returns

True/False Success/Fail or nil otherwise

Example
ScenEdit_UnloadCargo( '2cd64757-1b66-4609-ad56-df41bee652e5' )
