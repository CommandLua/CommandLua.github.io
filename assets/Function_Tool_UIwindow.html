Tool_UIwindow (name, mode)

This function opens or closes UI windows

Parameters

  • name UI Window The UI window
  • mode True/False Open or close boolean

Returns

True/False Opens the specified window if True, closes the window if False

Example
Tool_UIwindow("missioneditor", true)
